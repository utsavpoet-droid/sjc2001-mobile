# SJC2001 Mobile App Rules

- Follow the TDD.
- Use TypeScript everywhere.
- Keep mobile as a thin client.
- Do not duplicate backend business logic.
- Use Expo Router.
- Use React Query for server state.
- Use Zustand only for auth and local UI state.
- Keep code modular and production-ready.
- Do not add unnecessary dependencies.
- Before major edits, propose a plan.
- After changes, run lint and typecheck.
